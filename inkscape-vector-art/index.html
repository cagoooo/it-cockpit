STAGE 01

認識 Inkscape 介面 × 基本形狀工具

用「超級積木」拼出機器人頭像 — 一切都從圓形和方形開始

📐 完整操作步驟

  1. 點選左側工具列的 矩形與正方形 或 圓形/橢圓形/弧形 圖示。
  2. 在畫布上拖拉，同時按住 Ctrl 鍵，就能畫出完美的正方形與正圓形。
  3. 選擇黑箭頭「選取和變形物件」，點一下圖案可以縮放，再點一次可以旋轉。
  4. 使用「星形與多邊形工具」，上方工具列可切換規則多邊形 / 星形，調整頂角數量。
  5. 必學快捷鍵：Ctrl+滾輪 縮放畫面、Ctrl+S 存檔、Ctrl+Z 復原。
💡 五年級小提醒：新建檔案後第一個動作就是 Ctrl+S 存成 SVG 格式！這是 Inkscape 的「魔法原始檔」，下次才能繼續修改。

STAGE 02

填色、描邊與漸層 — 魔法調色盤

給圖形穿上五顏六色的衣服 + 酷炫的外框 + 水彩般的漸層

🎨 完整操作步驟

  1. 用黑箭頭選取圖形後，直接點擊畫面最下方的色票就能改變內部填色（Fill）。
  2. 按住 Shift 再點擊下方色票，就會改變描邊（Stroke）顏色。
  3. 打開「填充與邊框」面板：快捷鍵 Ctrl+Shift+F，可設定線條粗細、虛線樣式、透明度。
  4. 使用「漸層工具」：點選左側漸層工具圖示，在圖形上拖拉一條線，就會產生漸層！
  5. 清除填色點左下方的 X；清除描邊則是 Shift + 點 X。
⚠️ 踩雷區：圖形塗不上顏色或文字看不見？打開填充與邊框面板檢查最下方的透明度（Alpha）拉桿，如果被拉到 0 就會變透明！縮小物件後邊框變超粗？在上排工具列開啟「縮放物件時同時縮放邊框寬度」。

STAGE 03

路徑與節點 — 像捏黏土一樣自由

節點是「連連看小圓點」，路徑是連接它們的線條，可以自由變形！

✏️ 完整操作步驟

  1. 點選左側「繪製貝茲曲線及直線」（像鋼筆的工具）。畫直線：點一下放開、點下一個點放開。
  2. 畫曲線：點下去後不放開滑鼠並拖拉，控制桿會拉出曲線弧度。
  3. 結束繪製：按 Enter 鍵或滑鼠右鍵，線條就會切斷！
  4. 節點編輯：點選左側「用節點編輯路徑」工具（白箭頭），拖拉節點或控制桿改變形狀。
  5. 形狀轉路徑：選取後按 Ctrl+Shift+C，把形狀變成可編輯的節點路徑。
  6. 複製：Ctrl+C → Ctrl+V 貼游標位置；Ctrl+D 再製貼原位置（最常用！）。
🪄 為什麼貝茲曲線停不下來？因為電腦不知道你畫完了沒！記得用 Enter 或滑鼠右鍵告訴它「完成」。物件一直被磁鐵吸住亂跳？按 Shift+5 關閉「貼齊」功能。

STAGE 04

物件操作 — 對齊、群組與布林運算

布林運算 = 圖形的「剪刀與膠水」，把簡單形狀組合成全新作品！

🍪 三大布林運算（餅乾模具大法）

  1. 相加 / 聯集（Union）：兩塊黏土揉成一塊大黏土。【路徑】→【相加】或 Ctrl++
  2. 減去 / 差集（Difference）：拿星星模具壓麵團，挖一個星星洞！放最上層的當餅乾模具。【路徑】→【減去】或 Ctrl+-
  3. 交集（Intersection）：只保留兩個圖形重疊的部分。【路徑】→【交集】或 Ctrl+*
  4. 對齊與分布：框選多物件 → 【物件】→【對齊與分佈】或 Shift+Ctrl+A
  5. 群組：Ctrl+G 把多物件綁在一起；Ctrl+Shift+G 解散。
⚠️ 布林運算沒效果？圖形可能是「群組」或「文字」（不是路徑），要先解散群組（Ctrl+Shift+G）或文字轉路徑（Ctrl+Shift+C）。減去結果不對？檢查「要當餅乾模具的圖形是否在最上層」。

STAGE 05

文字工具與圖層管理 — 透明便利貼疊出風景

圖層像疊在一起的透明便利貼 + 文字工具讓字繞著圓圈跳舞！

📝 完整操作步驟

  1. 點左側「A」文字工具，在畫布上「點一下」（不要拖曳！）就能打字。
  2. 在上方工具列調整字型、大小、粗體、字間距。Inkscape 支援所有中文字型！
  3. 打開圖層面板：Ctrl+Shift+L。點 + 新增圖層，雙擊圖層名稱可改名。
  4. 鎖定圖層：點「鎖頭圖示」；隱藏圖層：點「眼睛圖示」。
  5. 超酷特效「置於路徑」：先打文字 → 畫一個圓 → 同時選取兩者 →【文字】→【置於路徑】，文字就會沿著圓彎曲！
  6. 右鍵物件 → 移到圖層 → 選擇目標圖層，可在圖層間搬移物件。
⚠️ 文字打不出來變空白？(1) 是「拖拉」開啟文字框，正確是「點擊」；(2) 字型不相容，試換字型；(3) 白字配白底，改顏色看看。記得養成每個圖層都改名字的好習慣！

STAGE 06

匯出 SVG 與 PNG — 把作品變成可分享的圖片

SVG = 魔法原始檔（可重編輯）；PNG = 平面貼紙（可貼簡報、支援透明）

💾 完整操作步驟

  1. 第一步永遠是存原始檔：【檔案】→【儲存】或 Ctrl+S，存成 .svg 格式。
  2. 匯出 PNG：【檔案】→【匯出 PNG 圖片】或 Ctrl+Shift+E。
  3. 選對匯出範圍很重要：
    • 頁面：匯出整個畫布（最常用）
    • 繪圖：匯出畫布外所有東西
    • 選取範圍：只匯出選取物件（適合去背貼紙！）
  4. 設定尺寸（解析度 96 DPI 即可）、檔名、按【匯出】。
  5. PNG 最大特色：支援透明背景！沒選取的部分會是透明，貼到簡報不會有白色方塊。
💡 SVG vs PNG 怎麼選？ 還要再修改 → 選 SVG！要貼到 Word / Google 簡報 / LINE 貼圖 → 選 PNG（記得選「選取範圍」做去背）。檔名後出現 * 代表還沒存檔，要趕快 Ctrl+S！

❓ 常見問題 FAQ

Q1：Inkscape 真的免費嗎？跟 Illustrator/Photoshop 有什麼不同？
✅ 完全免費！由全世界一群熱愛畫畫的人寫程式免費送給大家用的。Photoshop 是畫「點陣圖」的（放大會變馬賽克），Inkscape 跟 Illustrator 同屬向量陣營，圖案放大幾百倍都還是清晰平滑。Illustrator 一個月要 300 多元，Inkscape 永遠免費！
Q2：為什麼老師說畫完要存成 SVG，不是 JPG 或 PNG？
因為 SVG 是「魔法原始檔」！電腦會記住你畫的所有圓形、方形、圖層，明天打開還能繼續修改。存成 JPG/PNG 就變成「平平的貼紙」，不能拆開修改。每次畫完一定要先存 SVG 當秘密配方！
Q3：「路徑」跟「形狀」有什麼不一樣？
用工具直接拉出來的星星、圓形、方形叫「形狀」，有專屬控制把手（可調星星角數）。按 Ctrl+Shift+C 變成「路徑」後，就會充滿節點可以像黏土一樣自由捏，但會忘記自己曾是顆星星，不能再用工具改角數了！
Q4：複製貼上後圖案跑到旁邊去了？
`Ctrl+C` → `Ctrl+V` 會貼在「滑鼠目前的位置」。要原地複製請用魔法快捷鍵 Ctrl+D（再製）！新圖案會一毫不差誕生在原物件正上方，做影子或重複圖案最方便。
Q5：畫錯了怎麼還原？怎麼一次復原很多步驟？
按 Ctrl+Z 就回到上一步！亂點五下就連按五次，像時光倒流退回去。如果退太多了，按 Ctrl+Shift+Z（取消復原）就回到未來。
Q6：要怎麼貼到 Word 或 Google 簡報裡？
Word 和 Google 簡報不一定看得懂 SVG，要匯出成 PNG：【檔案】→【匯出 PNG 圖片】。PNG 的超能力是支援「透明背景」，小恐龍貼到簡報不會有醜醜的白色方塊，完美融入背景！
