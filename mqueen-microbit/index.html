STAGE 01

📡 認識 micro:bit 大腦

micro:bit 是英國 BBC 設計、比信用卡還小的「電子大腦」。它一片就有 25 顆 LED 燈、A/B 兩顆按鈕、加速度計、磁力計（指南針）、無線電、藍牙。把它插上麥昆小車，就成為機器人的「中央處理器」。

🔍 micro:bit 的六大超能力（用比喻記）

  • 125 LED 點陣：機器人的「表情看板」，可顯示愛心、笑臉、文字訊息
  • 2A/B 按鈕：兩顆「神奇開關」，像遊戲手把按鍵
  • 3加速度計：「內耳平衡神經」，知道板子有沒有傾斜搖晃
  • 4磁力計：「內建指南針」，知道方位東南西北
  • 5無線電 (Radio)：「對講機」，多塊 micro:bit 可以互相通訊
  • 6藍牙 (Bluetooth)：「隱形傳輸線」，連接手機 / 平板
💡 阿凱老師提示micro:bit V2 比 V1 多了「麥克風」和「喇叭」喔，可以自己發聲音！

🧩 MakeCode 圖形化積木編輯器

MakeCode 是微軟開發的免費線上程式編輯器，不用安裝、打開瀏覽器就能用。寫程式就像玩樂高積木，拖曳拼湊就能完成。

🚀 第一個程式「姓名徽章」操作步驟

  • 1打開瀏覽器，輸入 makecode.microbit.org
  • 2點選「新增專案」，認識三大區塊：左：模擬器、中：積木工具列、右：編輯區
  • 3從「基本」拖出「重複無限次」積木到編輯區
  • 4放入「顯示文字」積木，輸入自己的名字
  • 5用 USB 線連接 micro:bit，電腦會跳出名為 MICROBIT 的隨身碟
  • 6點「下載」→ 把 .hex 檔拉進 MICROBIT 隨身碟。完成！🎉
⚠️ 注意下載時 micro:bit 背面的黃燈會閃爍，不可以拔掉 USB 線，要等閃爍結束。

Q2. 用 MakeCode 寫好程式後，要怎麼把它「燒進」micro:bit？
STAGE 03

🚗 麥昆小車組裝啟動

如果 micro:bit 是大腦，那麥昆小車就是聽從大腦指揮的身體！它身上有馬達（雙腿肌肉）、循跡感測器（眼睛）、超音波模組（聲納）、RGB 燈（霓虹底盤）、蜂鳴器（嘴巴）、紅外線接收頭（順風耳）。

🔧 組裝四步驟

  • 1裝電池：打開電池盒，裝入 3 顆 AAA (7號) 電池，注意正負極（電源開關必須在 OFF）
  • 2插超音波：把兩顆「大眼睛」（SR04）插在車頭專屬插槽，T 對 trig、E 對 echo
  • 3插 micro:bit：金手指朝下，正面（LED 矩陣）朝前方，插入小車黑色插槽
  • 4開電源：打開車尾電源開關（切到 ON），看小車是否亮起電源指示燈
⚠️ 安全提醒裝電池或插拔感測器時，務必先把電源切到 OFF，避免短路！

Q3. 麥昆小車已經裝好電池、micro:bit 也插好了，但車子完全不動，最可能是什麼原因？
STAGE 04

⚡ 馬達控制（讓小車動起來）

馬達是小車的「雙腿肌肉」。MakeCode 的馬達積木可以分別控制左輪、右輪、或全部，方向選擇前進 / 後退，速度範圍 0~255。兩輪速度差就是轉彎的關鍵！

🚦 操作步驟

  • 1拖出「重複無限次」積木
  • 2放入「馬達 (全部) 方向 (前進) 速度 (100)」
  • 3加「暫停 (毫秒) 1000」（前進 1 秒）
  • 4改成「左馬達 速度 0、右馬達 速度 100」→ 小車左轉
  • 5練習：用 A 鍵控制前進、B 鍵後退、A+B 停止
💡 馬達校正如果小車直走會偏掉（兩顆馬達摩擦力不同），可以調整左右輪速度差，例如左 255、右 240 來修正。

STAGE 05

📏 循跡感測（沿著黑線跑）

車底有 L1、R1 兩顆紅外線感測器，像「永遠盯著地板看的眼睛」。它分辨黑與白：黑線輸出 0、白色輸出 1。靠這兩顆眼睛 + 「如果...那麼...否則」邏輯判斷，車子就能像火車一樣沿著黑線跑！

🧠 循跡邏輯（4 種情況）

  • 1L1=0 且 R1=0（兩顆都壓黑線）→ 馬達全前進
  • 2L1=0 且 R1=1（左壓黑、右偏白）→ 左馬達停、右馬達前進（向左修正）
  • 3L1=1 且 R1=0（右壓黑、左偏白）→ 左馬達前進、右馬達停（向右修正）
  • 4L1=1 且 R1=1（兩顆都白）→ 跑出黑線了，停止或慢慢轉
📐 場地準備用白色大圖畫紙 + 黑色絕緣膠帶貼軌道。膠帶寬度要夠，讓兩個感測器能同時感測到黑線。

STAGE 06

📐 超音波避障（不撞牆）

超音波模組就像蝙蝠的聲納雷達：發出聲波→撞到障礙物彈回→計算距離。設定一個安全距離（例如 30 公分），距離小於 30 就煞車轉彎，這就是自動駕駛車「自動煞車系統 (AEB)」的雛型！

🛑 程式邏輯

  • 1建立變數 distance
  • 2在「重複無限次」中：distance = 讀取超音波 (CM)
  • 3如果 distance < 30 → 馬達停 → 馬達右轉 → 暫停 800 毫秒
  • 4否則（安全前進）→ 馬達全前進
⚠️ 常見坑超音波偶爾會跳到 0 公分（反射錯誤）。寫程式時加條件「distance ≠ 0 且 < 30」，並加 100ms 延遲防止錯誤訊號。

📱 無線遙控（紅外線 + 藍牙）

麥昆有兩種無線遙控方式：紅外線遙控器（像電視遙控器）+ 藍牙手機/平板 App（用 Gamepad 搖桿）。

🎮 紅外線遙控（按鍵代碼對照）

  • 1「當啟動時」設定紅外線接收引腳
  • 2拖「當紅外線接收到 message」積木（自動建立 message 變數）
  • 3對應按鍵代碼：2 前進、8 後退、4 左轉、6 右轉、5 停止
💡 遙控器有兩種規格KEYES 與 NEC 解碼格式不同。寫前先測試一下：寫一段「顯示 message」程式，按下遙控器看跑出什麼數字，再填入對應指令。

STAGE 08

🚓 警車防撞綜合專題（實戰挑戰）

最終 Boss 戰！綜合「超音波避障 + 警報音效 + 紅藍車燈閃爍」做出真正的智慧警車：平常巡邏前進，遇到障礙物就煞車轉彎、發出警報並閃紅藍燈！

🎬 完整邏輯

  • 1設定 distance 讀取超音波
  • 2如果 distance < 30（遇到障礙）：
    • → 馬達後退 + 隨機左/右轉
    • → 蜂鳴器播放警報音效（從低音爬升到高音）
    • → 車燈/RGB 紅色閃爍
  • 3否則（安全前進）：
    • → 馬達全前進
    • → 停止音效
    • → RGB 改為綠色（巡邏中）
🔊 蜂鳴器開關麥昆側邊有「蜂鳴器開關」，必須切到 ON 才會發聲！蜂鳴器和 P0 共用，使用 P0 時要關閉蜂鳴器。

▼ FAQ：學生最常問的問題

Q1. micro:bit 和 Arduino 有什麼不一樣？我該學哪個？
micro:bit 是「內建超多裝備的超級大腦」（自帶 25 顆 LED、2 顆按鈕、加速度計、磁力計、藍牙、無線電），不用拉線焊接就能玩；Arduino 像「什麼都沒有的空板」，要自己接線焊接。對國小初學者，micro:bit 絕對是最好玩、最不容易失敗的選擇！
Q2. MakeCode 要付錢嗎？回家可以用嗎？
完全免費！ MakeCode 是微軟（Microsoft）開發的免費線上軟體，不需要在電腦安裝任何程式，只要打開瀏覽器（Chrome、Edge）輸入 makecode.microbit.org 就能開始寫程式。回家只要有網路，隨時都能繼續創作！
Q3. 螢幕旁邊那個會動的「模擬器」有什麼用？
模擬器是你的「虛擬試車場」！在把程式真正下載到實體 micro:bit 之前，可以先在畫面左邊看虛擬大腦的 LED 燈、按鈕有沒有照預期運作。確認沒寫錯再下載到車子裡，可以幫你省下大量除錯時間！
Q4. 麥昆小車沒電了怎麼辦？要怎麼充電？
「麥昆 Lite」基礎版用 3 顆 AAA (7號) 電池，沒電就換新電池（注意正負極，安裝時電源開關必須在 OFF）。「麥昆 Plus V2」用 18650 鋰電池，可以用 USB 線直接充電。電壓低於 2V 時，板子 LED 還會亮但馬達跑不動，這時就是該換電池了！
Q5. 麥昆延伸庫安裝失敗怎麼辦？
最常見的解法：(1) 切換到 JavaScript 模式 → 點開左側 Explorer → 點 Maqueen 旁邊的「更新」按鈕（refresh 圖示）；(2) 清除瀏覽器 cookie 或改用「無痕模式」。實在不行可以直接貼網址 github.com/DFRobot/pxt-maqueen 加入。
Q6. 學會後可以做什麼專題？能參加比賽嗎？
超多應用！可以做相撲機器人、無人快遞車、警車巡邏、自動避障車、鋼琴小車。比賽方面可以挑戰 「2025 科技寶國際機器人競賽」國小組（迷宮避障、自動搬運），或是各縣市的創客比賽。家裡也可以買一套來繼續玩，麥昆 Lite 全套約 1500-2000 元，CP 值很高！
Q7. 馬達會不會夾到手？麥昆容易壞嗎？
非常安全！麥昆有 ABS 塑膠外殼包覆齒輪，不像有些機器人齒輪外露；電路板還有「防反接保護」，就算電池裝反也不會燒板子。但畢竟是精密電子儀器，不要當碰碰車摔，也不要放在有水的地方。好好愛護可以陪你玩好多年！
Q8. 藍牙配對失敗怎麼辦？
最常見原因是「30 秒超時」。重新按 A+B+Reset 進入配對模式，30 秒內在 App 點亮跟板子上一模一樣的 5×5 圖案。建議改用小車電池供電（USB 供電有時電力不穩）。
